# Input Examples for Every Supported Type

Use any of these with the CLI, REST API, or Streamlit UI.

```bash
python -m src.cli analyze "<example>"
python -m src.cli evidence "<example>"      # no LLM call
```

---

### Quick verification

```bash
# Fast (no LLM)
python -m src.cli evidence "EGFR p.Leu858Arg"
python -m src.cli evidence "BRCA1 c.5074+1G>A"
python -m src.cli evidence "BCR::ABL1"
python -m src.cli evidence "EGFR L858R; T790M"

# Full pipeline (uses NVIDIA_API_KEY)
python -m src.cli analyze "BRAF V600E"
python -m src.cli analyze "HER2 amplification" --md her2.md
python -m src.cli analyze "MET exon14skip" --model nvidia/llama-3.1-nemotron-70b-instruct
```

The parser normalises everything to a canonical form before fetching evidence: `p.Val600Glu` → `V600E`, `HER2` → `ERBB2`, `EML4-ALK` → `EML4::ALK fusion`, etc.
